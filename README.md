# 🛒 eCommerce Website (Frontend)

A responsive eCommerce website built using HTML, CSS, and JavaScript.  
This project includes product listings, navigation menu, cart interface, and modern UI design.

## 🚀 Features
- Responsive navigation bar with toggle menu
- Product listing pages
- Product detail page
- Shopping cart UI
- Categories and featured products
- Testimonials and brands section
- Footer with useful links
- Interactive menu using JavaScript

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Clone or download the project
2. Open `secondproject.html` in your browser
